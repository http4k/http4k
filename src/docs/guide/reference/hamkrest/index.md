### Installation (Gradle)

```kotlin
dependencies {
    implementation(platform("org.http4k:http4k-bom:5.23.0.0"))
    implementation("org.http4k:http4k-testing-hamkrest")
}
```

### About

A set of Hamkrest matchers for use when testing http4k apps.
