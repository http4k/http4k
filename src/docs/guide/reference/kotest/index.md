### Installation (Gradle)

```kotlin
dependencies {
    implementation(platform("org.http4k:http4k-bom:5.11.1.0"))
    implementation("org.http4k:http4k-testing-kotest")
}
```

### About

A set of [Kotest] matchers for use when testing http4k apps.
