### Installation (Gradle)

```kotlin
dependencies {
    implementation(platform("org.http4k:http4k-bom:5.8.0.0"))
    implementation("org.http4k:http4k-testing-strikt")
}
```

A set of Strikt matchers for use when testing http4k apps.
