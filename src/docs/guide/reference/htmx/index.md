### Installation (Gradle)

```kotlin
dependencies {
    implementation(platform("org.http4k:http4k-bom:5.7.1.0"))
    implementation("org.http4k:http4k-htmx")

    implementation("org.http4k:http4k-handlebars") // for example
}
```

### About

Utilities to support [HTMX](https://htmx.org) development. Includes the Webjar and a set of classes/functions to ease development of HTMX powered applications.
